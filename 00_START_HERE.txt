🎯 QUICK START (3 minutes):
═══════════════════════════════════════════════════════════════════════

1. Go to folder:
   $ cd tohen-main

2. Create config:
   $ cp .env.example .env

3. Edit config (add your API keys):
   $ nano .env
   
   • ANTHROPIC_API_KEY=sk-...
   • GOOGLE_API_KEY=...
   • OPENAI_API_KEY=sk-...

4. Start services:
   $ ./start.sh start

5. Open browser:
   http://localhost:3000
